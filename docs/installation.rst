Installation
============

Requirements:

* Django >=2.1
* Python 3.5, 3.6, 3.7


To install::

    pip install django-permissions-auditor


Add `permissions_auditor` to your ``INSTALLED_APPS`` in your project's ``settings.py`` file::

    INSTALLED_APPS = [
        ...
        'permissions_auditor',
        ...
    ]
